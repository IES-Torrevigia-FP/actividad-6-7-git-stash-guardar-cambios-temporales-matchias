La diferencia entre git stash normal y git stash -u es que el comando normal solo guarda los cambios en archivos que ya están siendo seguidos por Git. git stash -u también guarda los archivos que aún no están siendo seguidos por Git permitiendo recuperar después esos archivos temporales junto con los demás cambios.
