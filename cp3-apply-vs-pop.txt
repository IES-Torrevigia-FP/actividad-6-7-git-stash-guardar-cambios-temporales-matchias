La entrada de stash se guardó en caso de ser necesario nuevamente.

La diferencia entre git stash apply y git stash pop es que:

git stash apply aplica los cambios guardados en el stash pero no elimina el stash, por lo que se puede volver a usar después.
git stash pop aplica los cambios y elimina el stash automáticamente de la lista.

Esto significa que apply es útil si quieres reutilizar el stash o probar los cambios, mientras que pop se usa cuando ya no necesitas guardar ese stash
